# Prompt Rules - 日报生成累积规则

> 本文件用于记录日报生成过程中发现的问题和改进规则。
> 脚本每次运行时会自动读取本文件内容，追加到 Prompt 末尾作为额外约束。
> 请按时间倒序添加新规则（最新的放最上面），格式不限。

---

## 使用方法
- 每次看完日报后，如果发现问题或有改进想法，直接编辑本文件添加规则
- 规则会在下次生成日报时自动生效
- 建议每条规则标注日期，方便追溯
